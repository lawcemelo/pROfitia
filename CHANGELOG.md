# Changelog

pROfitia の主な変更履歴です。

## [1.0.5] - 2026-06-13

### Added

- MD明細モーダルに行ごとの計算アシストと列ヘッダを追加しました。
- MD管理タブの表示ショートカットを `Alt + R + O` / `Shift + R + O` / `Ctrl + R + O` に対応しました。

### Changed

- MD管理タブを開いたときは週間モードへ自動切替し、MD管理中は月間モードへ切り替わらないようにしました。
- MD明細モーダルの横幅、アイテム入力欄、計算ボタン配置を調整しました。
- MD管理グリッドの列ヘッダを縦スクロール時にも固定表示するようにしました。

### Fixed

- 月間モード時のMD管理操作を抑止し、週間モード案内を表示するようにしました。
- MD管理タブ表示ショートカットが環境差で効かない場合がある問題を修正しました。
- 収支確認のMD別収支で、同名MD行の明細展開が別パネルと連動する問題を修正しました。
- 明細検索の日付範囲区切りの文字化けを `～` に修正しました。

## [1.0.4] - 2026-06-12

### Added

- MD管理タブを追加し、MDごとの周回状況をキャラクター別またはMD別に確認できるようにしました。
- MD完了時の左クリックチェック、右クリックからの明細付きチェック登録に対応しました。
- MD明細登録モーダルを追加し、攻略時間、開始時間からの経過時間、複数明細入力に対応しました。
- キャラクターマスタを追加し、名前、job、レベル、アイコン画像のアップロードと切り抜き登録に対応しました。
- MDマスタを追加し、MD名、ID名、リセット種別、条件レベル、非表示設定を登録できるようにしました。
- タグMD反映機能を追加し、指定タグが付いた登録済データを選択したMDへ一括反映できるようにしました。
- 収支確認にMD別集計、MD別収支グラフ、MD別時給収支、MD収支軸設定を追加しました。
- Ragnarok画面を監視してキャラクター名とMD IDを検出するTesseract前提の監視アプリを追加しました。

### Changed

- 設定タブのマスタ関連機能を「グループ管理」と「MD管理」の展開式インタフェースに整理しました。
- MD管理の表示を、選択中の週間または月間に応じた状態で反映するようにしました。
- MD管理のキャラクター行、MD行を切り替え可能にし、行ヘッダまたは列ヘッダのドラッグで並び替えできるようにしました。
- MD管理の開始時間セットを上部説明文と並列に配置し、経過時間をリアルタイム表示するようにしました。
- 明細入力、明細編集、予定編集、メモ編集、検索、色テーマなどの文言と配置を調整しました。
- 収支確認グラフの棒グラフと折れ線グラフを、始点から終点まで描画されるアニメーション表示にしました。
- MD管理、MD明細登録、MD収支軸登録の主要登録ボタン配置を右下基準へ調整しました。

### Fixed

- MD管理のチェックと明細の紐づけを見直し、明細付きチェック解除時に確認を出すようにしました。
- MD管理の列幅、固定列、折り返し、アイコン表示、グレーアウト表示、チェックボタン周辺の見た目を調整しました。
- トップヘッダの週間/月間切り替え、期間選択、前後移動ボタンの位置ずれを修正しました。
- MDマスタ、キャラクターマスタ、タグMD反映、メモ編集などの入力コントロール高さや横幅の不揃いを修正しました。
- 数値入力で全角数字や `１００ｋ` のような表記をEnter確定時に正規化するようにしました。
- 画面内に残っていた文字化け、英字混在、ラベル不足を修正しました。

## [1.0.3] - 2026-05-18

### Changed

- サムネイル画像を白背景の `1200x800` 画像に再生成しました。
- トップサマリーの折りたたみボタン文言を、状態が分かりやすい表示に変更しました。

### Fixed

- 明細入力の時間欄が非表示の場合、追加時刻の現在時刻で登録されるように修正しました。
- 明細入力の時間欄が表示中の場合、入力した時刻で登録されるように修正しました。

## [1.0.2] - 2026-05-16

### Added

- 予定管理に数量入力を追加しました。
- 予定編集画面に数量、予定金額、削除機能を追加しました。
- 予定一覧に左端ドラッグハンドルを追加しました。
- 予定一覧に先頭/末尾へ移動できるボタンを追加しました。
- 清算モードの人数入力にカウントアップ/ダウンボタンを追加しました。

### Changed

- 予定管理の予定額を予定単価として扱い、数量 x 予定単価 = 予定金額で表示するように変更しました。
- 予定一覧の表示を、アイテム名の下に単価、数量、タグを表示する形式に変更しました。
- 予定金額を売却は青系、購入は赤系で表示するように変更しました。
- 予定管理の累計差引額の幅と表示を調整しました。
- 明細追加/明細編集のグループ選択見出しを「グループから選択する場合」に変更しました。
- `guide.html` に操作イメージを追加し、予定管理などの説明を現行仕様に更新しました。

### Fixed

- 予定から明細へ送る際、登録数量が予定数量より少ない場合に差分数量を予定へ残すように修正しました。

## [1.0.1] - 2026-05-15

### Added

- 設定タブのバックアップ内にデータのオールリセット機能を追加しました。
- URL共有時のサムネイル表示に対応し、OGP/Twitter Card設定を追加しました。
- サムネイル画像 `assets/ogp.png` を追加しました。

### Changed

- 設定タブ左メニューにオールリセットを追加しました。
- 設定タブ左メニューの配置とデザインを調整しました。
- 説明ページ `guide.html` の機能説明を現行仕様に合わせて整理しました。
- 説明ページ内の具体的なアイテム例を仮名称に変更しました。

### Fixed

- 全角 `＃` で入力したタグもタグチップ化されるように修正しました。

## [1.0.0] - 2026-05-15

### Added

- 明細入力、清算、予定管理、明細検索、収支確認、メモ、設定の基本機能を追加しました。
- 週間/月間の収支確認、目標収支、タグ別/アイテム別/項目別の明細分析を追加しました。
- グループ、アイテム、グループ構成、色テーマ、バックアップの設定機能を追加しました。
- 説明ページ `guide.html`、サイトアイコン、タイトルロゴ、`LICENSE` を追加しました。

### Changed

- タブ名を「予定入力」から「予定管理」に変更しました。
- 明細入力と収支確認のサマリーを、件数ではなく金額合計ベースの表示に変更しました。
- README を公開用の説明に更新しました。

### Notes

- データは各ユーザーのブラウザ内に保存されます。
- ブラウザデータ削除や端末変更に備えて、定期的なエクスポートを推奨します。
