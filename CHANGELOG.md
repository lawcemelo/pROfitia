# Changelog

pROfitia の主な変更履歴です。

## [1.1.3] - 2026-07-15

### Added

- MD明細登録と清算登録で、攻略人数/清算人数に応じた1人当たり金額を保存・集計できるようにしました。
- アイテムマスタに公式IDを登録し、公式ツールページを開けるようにしました。
- 設定タブ内の各設定パネルを `+` / `-` ボタンで折りたたみ・展開できるようにしました。

### Changed

- 明細入力の清算モードでは「MD明細から入力」ボタンを非表示にしました。
- 収支確認のMDグラフに合計収支の棒グラフと専用軸登録を追加し、合計収支軸を左、MD収支軸を右に配置しました。
- MDグラフの合計収支棒の幅、角丸、枠線、凡例との余白を調整しました。

- 明細入力フォームのMD選択を廃止し、日付欄の上に「MD明細から入力」ボタンを追加しました。
- MD明細追加でMDを選択したとき、MD構成に紐づくアイテムを明細行へ反映するようにしました。

- 設定タブ内のアイテム入力を上部に配置し、アイテム追加フォームと一覧のレイアウトを調整しました。
- アイテムマスタ一覧をアイテム名昇順で表示し、公式ページを開ける行のボタン色を変更しました。
- MD管理グリッドのセル表示を見直し、右上の周回数表示を削除してチェック表示を中央配置に調整しました。
- MD別収支の明細展開で、人数割りがある場合は単価表示を `1,000 / 2人` の形式に変更しました。

### Fixed

- 清算登録・編集時に数量を変えず、1人当たり金額として明細保存されるように修正しました。
- MD明細登録時の金額計算が、攻略人数で割った1人当たり金額を参照するように修正しました。

## [1.1.2] - 2026-07-01

### Added

- MD管理グリッドで、キャラクターまたはMDのヘッダをクリックしたときに対象の行列を強調表示し、両方を選択した場合は交差セルも分かるようにしました。
- MD別収支に周回数、MD別時給収支に合計時間と時間/周を表示するようにしました。

### Changed

- MD管理タブの位置を、明細入力と予定管理の間へ移動しました。
- 収支確認の期間明細選択を、トップの週間/月間期間変更に追従するようにしました。
- 収支確認の期間一覧で、収支0の現在期間が前後期間へ移動しても残るようにしました。
- MD管理の行列選択中にチェック操作をしても、選択状態を維持するようにしました。
- MD別収支とMD別時給収支の列、収支バー、合計表示の見え方を調整しました。

## [1.1.1] - 2026-06-28

### Added

- 収支確認のMD別収支に合計金額を表示しました。

### Changed

- 収支確認のMDグラフ凡例を広げ、チェックなしの系列はグラフと凡例を薄く表示するようにしました。
- 収支確認のMD別収支バーを、マイナス前提の中央基準ではなく左起点のバー表示に変更しました。

## [1.1.0] - 2026-06-26

### Added

- 収支確認のMD表示で、MD別収支とMD別 時給収支に列ヘッダを追加しました。
- 収支確認のMD表示で、MD別収支とMD別 時給収支の列ヘッダからソートできるようにしました。
- 収支確認グラフの表示数を、週間は週数、月間は月数として任意指定できるようにしました。

### Changed

- 収支確認グラフを、ヘッダで選択中の週間または月間を終点にして表示するようにしました。
- 設定のタグ/グループMD反映で、MD選択ボックスの未選択テキストを「MDを選択」に変更しました。

## [1.0.9] - 2026-06-22

### Fixed

- 明細編集で支出を開いた際に、販売価格欄が表示される問題を修正しました。
- 明細編集の収入/支出切り替えで、使用しない単価入力が計算やバリデーションに混ざらないようにしました。

## [1.0.8] - 2026-06-15

### Changed

- MD管理の全画面表示で、MD周回表パネルが画面最下部まで広がるように調整しました。
- 収支確認の全体表示でも、タグ別収支とアイテム/項目別収支の明細展開に対応しました。
- 収支確認のMD表示の明細展開行を、全体表示側と同じ見え方に揃えました。

## [1.0.7] - 2026-06-15

### Added

- MD管理タブに全画面表示を追加しました。

### Changed

- MD管理の全画面表示時はトップヘッダとフッタを非表示にし、ブラウザの左右幅いっぱいまでMD管理画面を広げるようにしました。
- 全画面表示中のMD周回表が画面の残り高さを使い切るように調整しました。

## [1.0.6] - 2026-06-14

### Added

- MDマスタにアイテムを紐づけるMD構成マスタを追加しました。
- MD明細モーダルの初期明細に、MD構成で紐づけたアイテムを表示できるようにしました。
- MD構成登録でアイテムマスタを入力途中に絞り込み選択できるようにしました。
- MD管理のSTART/STOP/RESET操作と、タブ別アイコン表示を追加しました。

### Changed

- タグMD反映を、タグまたはグループを選択してMDへ一括変換する機能に変更しました。
- タグ入力、グループ選択、MD選択、一括変換ボタンの配置を整理しました。
- MD管理の開始時間ラベルを時刻入力の左に配置し、上側余白を詰めました。

### Fixed

- MD管理の横スクロール時に固定列と明細セルの重なりが崩れる問題を修正しました。

## [1.0.5] - 2026-06-13

### Added

- MD明細モーダルに行ごとの計算アシストと列ヘッダを追加しました。
- MD管理タブの表示ショートカットを `Alt + R + O` / `Shift + R + O` / `Ctrl + R + O` に対応しました。

### Changed

- MD管理タブを開いたときは週間モードへ自動切替し、MD管理中は月間モードへ切り替わらないようにしました。
- MD明細モーダルの横幅、アイテム入力欄、計算ボタン配置を調整しました。
- MD管理グリッドの列ヘッダを縦スクロール時にも固定表示するようにしました。

### Fixed

- 月間モード時のMD管理操作を抑止し、週間モード案内を表示するようにしました。
- MD管理タブ表示ショートカットが環境差で効かない場合がある問題を修正しました。
- 収支確認のMD別収支で、同名MD行の明細展開が別パネルと連動する問題を修正しました。
- 明細検索の日付範囲区切りの文字化けを `～` に修正しました。

## [1.0.4] - 2026-06-12

### Added

- MD管理タブを追加し、MDごとの周回状況をキャラクター別またはMD別に確認できるようにしました。
- MD完了時の左クリックチェック、右クリックからの明細付きチェック登録に対応しました。
- MD明細登録モーダルを追加し、攻略時間、開始時間からの経過時間、複数明細入力に対応しました。
- キャラクターマスタを追加し、名前、job、レベル、アイコン画像のアップロードと切り抜き登録に対応しました。
- MDマスタを追加し、MD名、ID名、リセット種別、条件レベル、非表示設定を登録できるようにしました。
- タグMD反映機能を追加し、指定タグが付いた登録済データを選択したMDへ一括反映できるようにしました。
- 収支確認にMD別集計、MD別収支グラフ、MD別時給収支、MD収支軸設定を追加しました。
- Ragnarok画面を監視してキャラクター名とMD IDを検出するTesseract前提の監視アプリを追加しました。

### Changed

- 設定タブのマスタ関連機能を「グループ管理」と「MD管理」の展開式インタフェースに整理しました。
- MD管理の表示を、選択中の週間または月間に応じた状態で反映するようにしました。
- MD管理のキャラクター行、MD行を切り替え可能にし、行ヘッダまたは列ヘッダのドラッグで並び替えできるようにしました。
- MD管理の開始時間セットを上部説明文と並列に配置し、経過時間をリアルタイム表示するようにしました。
- 明細入力、明細編集、予定編集、メモ編集、検索、色テーマなどの文言と配置を調整しました。
- 収支確認グラフの棒グラフと折れ線グラフを、始点から終点まで描画されるアニメーション表示にしました。
- MD管理、MD明細登録、MD収支軸登録の主要登録ボタン配置を右下基準へ調整しました。

### Fixed

- MD管理のチェックと明細の紐づけを見直し、明細付きチェック解除時に確認を出すようにしました。
- MD管理の列幅、固定列、折り返し、アイコン表示、グレーアウト表示、チェックボタン周辺の見た目を調整しました。
- トップヘッダの週間/月間切り替え、期間選択、前後移動ボタンの位置ずれを修正しました。
- MDマスタ、キャラクターマスタ、タグMD反映、メモ編集などの入力コントロール高さや横幅の不揃いを修正しました。
- 数値入力で全角数字や `１００ｋ` のような表記をEnter確定時に正規化するようにしました。
- 画面内に残っていた文字化け、英字混在、ラベル不足を修正しました。

## [1.0.3] - 2026-05-18

### Fixed

## [1.0.2] - 2026-05-16

### Added

- 予定管理に数量入力を追加しました。
- 予定編集画面に数量、予定金額、削除機能を追加しました。
- 予定一覧に左端ドラッグハンドルを追加しました。
- 予定一覧に先頭/末尾へ移動できるボタンを追加しました。
- 清算モードの人数入力にカウントアップ/ダウンボタンを追加しました。

### Changed

- 予定管理の予定額を予定単価として扱い、数量 x 予定単価 = 予定金額で表示するように変更しました。
- 予定一覧の表示を、アイテム名の下に単価、数量、タグを表示する形式に変更しました。
- 予定金額を売却は青系、購入は赤系で表示するように変更しました。
- 予定管理の累計差引額の幅と表示を調整しました。
- 明細追加/明細編集のグループ選択見出しを「グループから選択する場合」に変更しました。
- `guide.html` に操作イメージを追加し、予定管理などの説明を現行仕様に更新しました。

### Fixed

- 予定から明細へ送る際、登録数量が予定数量より少ない場合に差分数量を予定へ残すように修正しました。

## [1.0.0] - 2026-05-15

### Added

- 明細入力、清算、予定管理、明細検索、収支確認、メモ、設定の基本機能を追加しました。
- 週間/月間の収支確認、目標収支、タグ別/アイテム別/項目別の明細分析を追加しました。
- グループ、アイテム、グループ構成、色テーマ、バックアップの設定機能を追加しました。
- 説明ページ `guide.html`、サイトアイコン、タイトルロゴ、`LICENSE` を追加しました。

### Changed

- タブ名を「予定入力」から「予定管理」に変更しました。
- 明細入力と収支確認のサマリーを、件数ではなく金額合計ベースの表示に変更しました。
- README を公開用の説明に更新しました。

### Notes

- データは各ユーザーのブラウザ内に保存されます。
- ブラウザデータ削除や端末変更に備えて、定期的なエクスポートを推奨します。
