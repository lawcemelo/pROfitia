# Changelog

pROfitia の主な変更履歴です。

## [1.0.2] - 2026-05-16

### Added

- 予定管理に数量入力を追加
- 予定編集画面に数量、予定金額、削除機能を追加
- 予定一覧に左端ドラッグハンドルを追加
- 予定一覧に先頭/末尾へ移動できる `⇈` / `⇊` ボタンを追加
- 清算モードの人数入力にカウントアップ/ダウンボタンを追加

### Changed

- 予定管理の予定額を予定単価として扱い、数量 × 予定単価 = 予定金額で表示するように変更
- 予定一覧の表示を、アイテム名の下に単価、数量、タグを表示する形式に変更
- 予定金額を売却は青系、購入は赤系で表示するように変更
- 予定管理の累計差引列の幅と表示を調整
- 明細追加/明細編集のグループ選択見出しを「グループから選択する場合」に変更
- `guide.html` に操作イメージを追加し、予定管理などの説明を現行仕様に更新

### Fixed

- 予定から明細へ送る際、登録数量が予定数量より少ない場合に差分数量を予定へ残すように修正

## [1.0.1] - 2026-05-15

### Added

- 設定タブのバックアップ内にデータのオールリセット機能を追加
- URL共有時のサムネイル表示に対応するOGP/Twitter Card設定を追加
- サムネイル画像 `assets/ogp.png` を追加

### Changed

- 設定タブ左メニューにオールリセットを追加
- 設定タブ左メニューの配置とデザインを調整
- 説明ページ `guide.html` の機能説明を現行仕様に合わせて精査
- 説明ページ内の具体的なアイテム例を仮名称に変更

### Fixed

- 全角 `＃` で入力したタグもタグチップ化されるように修正

## [1.0.0] - 2026-05-15

### Added

- 明細入力機能を追加
  - 収入、支出、清算の3モードに対応
  - 価格、数量、金額の入力と自動計算に対応
  - K、M、G、T の単位入力に対応
  - 計算アシスト機能を追加
- 清算機能を追加
  - 複数アイテムの一括入力に対応
  - 人数割りによる配布金額計算に対応
  - 買取明細と買取者別配布金額の確認に対応
  - 追加前の確認モーダルを追加
- 予定管理機能を追加
  - 可用資金、売却予定、購入予定の管理に対応
  - 予定一覧の並び替え、編集、削除に対応
  - 予定から明細登録画面への送信に対応
- 明細検索機能を追加
  - 期間、種別、タグ、アイテム/項目名による検索に対応
  - 検索結果の合計サマリとソートに対応
  - 検索結果が1アイテムの場合の単価推移グラフを追加
- 収支確認機能を追加
  - 週間、月間の切り替え表示に対応
  - 収入、支出、収支、目標収支のグラフ表示に対応
  - タグ別、アイテム/項目別の明細分布を追加
  - 軸上限設定に対応
- メモ機能を追加
  - タグ、タイトル、内容、完了状態、色付きメモに対応
  - メモ検索、ソート、編集、削除に対応
- 設定機能を追加
  - グループ、アイテム、グループ構成の登録、編集、削除に対応
  - 色テーマ切り替えに対応
  - エクスポート/インポートによるバックアップに対応
- 説明ページ `guide.html` を追加
- 画面内ヘルプを追加
- サイトアイコンとタイトルロゴを追加
- `LICENSE` を追加し、著作権表記を整備

### Changed

- タブ名を「予定入力」から「予定管理」に変更
- 明細入力の収支取引概要を件数ではなく数量合算に変更
- 収支確認のアイテム/項目別明細分布を件数ではなく数量合算に変更
- README を公開用の説明に更新
- 説明ページを現行機能に合わせて更新

### Notes

- データは各ユーザーのブラウザ内に保存されます。
- ブラウザデータ削除や端末変更に備えて、定期的なエクスポートを推奨します。
- 初版のため、今後の変更はこのファイルに追記します。
