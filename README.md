# pROfitia 収支管理支援アプリ

pROfitia は、RagnarokOnline のゲーム内売買、清算、予定資金、収支推移を管理するためのブラウザアプリです。

## 主な機能

- 明細入力
  - 収入、支出、清算の3モードで登録できます。
  - アイテム/項目、価格、数量、タグを入力して明細を追加します。
  - グループ構成を使うと、グループからアイテムを絞り込んで選択できます。
  - 清算モードでは複数アイテムをまとめて入力し、人数ごとの配布金額を確認できます。

- 予定入力
  - 可用資金、売却予定、購入予定、差引予定を管理できます。
  - 買いたいアイテムに対して、どの売却予定で資金を作るか確認する用途に向いています。

- 明細検索
  - 期間、収入/支出区分、タグ、アイテム/項目名で検索できます。
  - 検索結果から明細編集もできます。

- 収支確認
  - 週間、月間の収入、支出、収支を確認できます。
  - グラフ、期間別一覧、タグ別・アイテム別分布で傾向を確認できます。

- 設定
  - グループ、アイテム、グループ構成を管理できます。
  - 色テーマを選択できます。
  - エクスポート/インポートでバックアップと復元ができます。

## GitHub Pages で公開するファイル

リポジトリ直下に以下のファイルとフォルダを配置してください。

```text
index.html
app.js
styles.css
favicon.svg
lp.html
lp.css
assets/
README.md
```

`index.html` がアプリ本体です。GitHub Pages ではリポジトリのトップページとして自動表示されます。

`lp.html` は機能説明ページです。アプリ内の「説明」ボタンから新規ページで開きます。

## GitHub Pages の有効化手順

1. GitHub に新しいリポジトリを作成します。
2. 上記ファイル一式をリポジトリ直下へアップロードします。
3. GitHub のリポジトリ画面で `Settings` を開きます。
4. 左メニューの `Pages` を開きます。
5. `Build and deployment` の `Source` で `Deploy from a branch` を選びます。
6. Branch は `main`、フォルダは `/root` を選びます。
7. `Save` を押します。
8. 数分後に表示される GitHub Pages のURLへアクセスします。

## データ保存について

このアプリのデータは、GitHub Pages 側のサーバーには保存されません。

登録データは、利用者それぞれのブラウザ内に保存されます。そのため、同じURLでアクセスしても、他の人のデータは見えません。

## 重要な注意点

以下の操作をすると、保存データが消える可能性があります。

- ブラウザのCookieやサイトデータを削除する
- 閲覧履歴やストレージを削除する
- 別のブラウザや別端末で開く
- シークレットウィンドウで利用する

定期的に、設定タブのエクスポート機能でバックアップJSONを保存してください。

復元する場合は、設定タブのインポート機能からバックアップJSONを読み込んでください。

## 推奨運用

- 普段使うブラウザを固定する
- 大きな取引や清算を入力した後はエクスポートする
- 月1回程度はバックアップを保管する
- PCとスマホで同じデータを使いたい場合は、エクスポート/インポートでデータを移動する

## クラウド同期について

現時点では、ログイン機能やクラウドDBによる自動同期はありません。

スマホとPCで完全に同じデータを自動同期したい場合は、Firebase や Supabase などのクラウドDB、ログイン機能、ユーザーごとの保存設計が別途必要です。
