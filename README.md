# pROfitia 収支管理支援アプリ

pROfitia は、RagnarokOnline のゲーム内取引を想定したブラウザ向け収支管理支援アプリです。
明細入力、予定管理、明細検索、収支確認、メモ、設定を使って、売買・清算・キャッシュフロー・振り返りを管理できます。

## 主な機能

- 収入、支出、清算の3モードで明細登録
- 価格、数量、金額の管理と自動計算
- 清算時の人数割り、買取者別の配布金額確認
- 売却予定、購入予定、可用資金による予定管理
- 期間、区分、タグ、アイテム/項目名による明細検索
- 検索結果が1アイテムの場合の単価推移グラフ
- 週間・月間の収支確認
- タグ別、アイテム/項目別の明細分布
- ToDoや出来事を記録するメモ機能
- グループ、アイテム、グループ構成、色テーマ、バックアップ管理

## データ保存について

このアプリの入力データはクラウドDBではなく、利用者それぞれのブラウザ内に保存されます。
GitHub Pages 側のサーバーやリポジトリに、ユーザーの収支データが保存されることはありません。

以下の操作で保存データが消える可能性があります。

- ブラウザのサイトデータを削除する
- 履歴やストレージを削除する
- 別ブラウザ、別端末で開く
- シークレットウィンドウで利用する

定期的に設定タブからエクスポートし、バックアップJSONを保存してください。
